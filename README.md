# Sorteo de Equipos de Futbol para Netlify

Versión corregida con Netlify Functions modernas usando `type: module`.

## Estructura correcta

- public/index.html
- netlify/functions/_shared.js
- netlify/functions/state.js
- netlify/functions/join.js
- netlify/functions/reset.js
- netlify.toml
- package.json

## Importante

Los archivos de funciones deben quedar dentro de `netlify/functions`, no en la raíz del repositorio.
